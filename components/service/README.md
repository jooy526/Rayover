# Service Components (Mobile)

**사용자용 모바일 앱** 전용 컴포넌트입니다.

## 대상 플랫폼

- **Mobile 전용**
- 기준 해상도: 360px (default grid)
- 터치 인터페이스

## 포함 예시

모바일 환경에 최적화된 컴포넌트:

- Bottom Sheet
- Action Sheet
- Mobile Header
- Tab Bar (하단 탭)
- Pull-to-refresh
- Mobile Navigation
- 기타 터치/모바일 전용 UI

## 사용 규칙

- ✅ Service(사용자) 화면 작업 시 `/common` + `/service` 사용
- ❌ Admin 화면에 사용 금지
- ❌ 데스크탑 반응형 금지

## 파일 구조

각 컴포넌트는 하나의 `.md` 파일로 정의합니다.
