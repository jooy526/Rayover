# Admin Components (PC)

**관리자용 PC 웹** 전용 컴포넌트입니다.

## 대상 플랫폼

- **PC 전용**
- 마우스/키보드 인터페이스
- 밀도 높은 정보 집약형 UI

## 포함 예시

PC 관리자 환경에 최적화된 컴포넌트:

- Data Table
- Data Grid
- Breadcrumb
- Sidebar Navigation
- Dropdown Menu
- Tooltip (hover 기반)
- Context Menu
- Page Header (PC)
- 기타 관리자/데스크탑 전용 UI

## 사용 규칙

- ✅ Admin(관리자) 화면 작업 시 `/common` + `/admin` 사용
- ❌ Service(모바일)에 사용 금지
- ❌ 모바일 반응형 금지

## 파일 구조

각 컴포넌트는 하나의 `.md` 파일로 정의합니다.
